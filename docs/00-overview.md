# プロジェクト概要

## アプリの目的

自分の子供の成長を記録し、アルバムで振り返り、時期に合わせた必需品を AI から提案するアプリ。

## 核となる要素

1. **成長記録** … 身長・体重・発達メモ・マイルストーン（寝返り、歩き始めなど）
2. **アルバム** … 写真・動画の時系列管理、イベント単位での整理
3. **時期別必需品の AI 提案** … 月齢・年齢に応じた「今買うと良いもの」の提案
4. **AI 拡張** … 成長アドバイス、アルバム補助（タグ・キャプション・1年前の今日）、メモ補助、意味検索
5. **認証・共有** … ログイン、将来的に家族など他ユーザーの記録閲覧

## 技術スタック

| 項目 | 選定 |
|------|------|
| FE | TypeScript + Next.js (App Router) |
| BE | TypeScript（Next.js API Routes / または別 API サーバー） |
| DB | MySQL（Docker） |
| ORM | Prisma |
| AI | OpenAI ChatGPT API（必要に応じて Vision） |
| UI | 赤ちゃん・家族を感じる温かみ × 洗練されたデザイン |

## Phase 一覧

- **Phase 0** … 土台（リポジトリ構成、Docker MySQL、docs 骨子）
- **Phase 1 (MVP)** … 認証、子どもプロフィール、成長記録、アルバム、AI（必需品・成長アドバイス）
- **Phase 2** … AI 拡張（アルバムAI、メモ補助、意味検索）
- **Phase 3** … 家族共有（招待・権限・他ユーザー記録閲覧）

## 用語

- **子ども（Child）** … プロフィール単位。1ユーザーが複数持てる（兄弟対応）。
- **成長記録** … 身長・体重・マイルストーン・メモなど、日付に紐づく記録。
- **アルバム** … 写真・動画とメタデータ（日付・キャプション等）。

## UI 方針

- キーワード: 赤ちゃん・家族・温かみ、洗練。
- トーン: 優しい色（ベージュ・ソフトピンク・ミント・サンド）、角を少し丸く、写真が主役になる余白多め。
- 避けたい: 子供っぽすぎるイラストだらけ、ごちゃごちゃした情報密度。
